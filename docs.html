Titanium Documentation

Everything you need to get started — from installation to advanced settings.

1 Disable antivirus

Turn off your antivirus for the time of installation and use. Without this, Titanium won't be able to work with Roblox.

2 Extract the archive

Extract the downloaded archive to any folder you like.

3 Run Titanium

Launch titanium.exe. Press Attach to connect to Roblox and start using scripts.

4 Enjoy

Titanium is ready. Monaco Editor, ScriptHub, 8 themes — everything is available right away.
